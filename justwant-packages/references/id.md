# @justwant/id

ID generation. Subpath imports only.

## Subpaths

| Subpath | Usage |
|---------|-------|
| sortable | createId(), getTimestamp(id) — time-sortable (ULID) |
| short | createId(size?) — short URL-safe (nanoid) |
| primitives | ulid, nanoid, decodeTime, monotonicFactory, customAlphabet |
