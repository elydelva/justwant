## Install

```bash
bun add @justwant/waitlist
```

For referral integration, also install `@justwant/referral`:

```bash
bun add @justwant/waitlist @justwant/referral
```

## TypeScript

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2020",
    "moduleResolution": "bundler",
    "strict": true
  }
}
```
