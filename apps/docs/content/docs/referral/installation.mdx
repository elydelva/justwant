## Install

```bash
bun add @justwant/referral
```

For waitlist integration, also install `@justwant/waitlist`:

```bash
bun add @justwant/referral @justwant/waitlist
```

## TypeScript

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2020",
    "moduleResolution": "bundler",
    "strict": true
  }
}
```
