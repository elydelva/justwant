## Install

```bash
bun add @justwant/organisation
```

`@justwant/organisation` depends on `@justwant/membership` and `@justwant/permission`, which are installed automatically as transitive dependencies.

If you have not set up `@justwant/membership` and `@justwant/permission` services yet, install them explicitly so you can configure their repos:

```bash
bun add @justwant/organisation @justwant/membership @justwant/permission
```
