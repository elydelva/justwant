## Install

```bash
bun add @justwant/membership
```

`@justwant/membership` has no required runtime dependencies beyond `@justwant/meta`.

## TypeScript

```json title="tsconfig.json"
{
  "compilerOptions": {
    "target": "ES2020",
    "moduleResolution": "bundler",
    "strict": true
  }
}
```
