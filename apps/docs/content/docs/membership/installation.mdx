## Install

<CodeBlockTabs defaultValue="bun">
  <CodeBlockTabsList>
    <CodeBlockTabsTrigger value="bun">Bun</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="npm">npm</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="pnpm">pnpm</CodeBlockTabsTrigger>
    <CodeBlockTabsTrigger value="yarn">Yarn</CodeBlockTabsTrigger>
  </CodeBlockTabsList>
  <CodeBlockTab value="bun">

```bash
bun add @justwant/membership
```

  </CodeBlockTab>
  <CodeBlockTab value="npm">

```bash
npm install @justwant/membership
```

  </CodeBlockTab>
  <CodeBlockTab value="pnpm">

```bash
pnpm add @justwant/membership
```

  </CodeBlockTab>
  <CodeBlockTab value="yarn">

```bash
yarn add @justwant/membership
```

  </CodeBlockTab>
</CodeBlockTabs>

`@justwant/membership` has no required runtime dependencies beyond `@justwant/meta`.
