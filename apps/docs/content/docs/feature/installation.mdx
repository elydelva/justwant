## Define a feature

Create a file that exports your feature definitions so they can be shared across packages.

```ts title="lib/features.ts"
import { defineFeature } from "@justwant/feature";

export const billing = defineFeature({ name: "billing" });
export const analytics = defineFeature({ name: "analytics" });
```

Pass the result to any `@justwant` package that accepts a `FeatureDef`:

```ts title="lib/flags.ts"
import { defineFlag } from "@justwant/flag";
import { billing } from "./features";

export const billingFlag = defineFlag(billing, {
  default: false,
  rules: [],
});
```
