## Quick setup

Define your actor types once and reuse the factories across your application.

```ts title="lib/actors.ts"
import { defineActor } from "@justwant/actor";

// Simple actor — identified by a single id
export const userActor = defineActor({ name: "user" });

// Scoped actor — identified by a withinId and an id
export const memberActor = defineActor({ name: "member", within: "org" });
```

## Create actor instances

```ts title="app/example.ts"
import { userActor, memberActor } from "@/lib/actors";

// user:user_123
const user = userActor("user_123");

// member:mbr_456 scoped inside org:org_789
const member = memberActor("org_789", "mbr_456");
```
