## Overview

The SMS channel uses [Twilio](https://twilio.com) for SMS delivery. You pass your Twilio client instance to the adapter.

## Install peer dependency

```bash
bun add twilio
```

## Import

```ts
import { createTwilioSmsCanal } from "@justwant/notify/channels/sms/twilio";
```

## Setup

```ts title="lib/notify.ts"
import { createNotify, createTemplate } from "@justwant/notify";
import { createTwilioSmsCanal } from "@justwant/notify/channels/sms/twilio";
import twilio from "twilio";

const smsCanal = createTwilioSmsCanal({
  client: twilio(process.env.TWILIO_ACCOUNT_SID!, process.env.TWILIO_AUTH_TOKEN!),
  from: process.env.TWILIO_PHONE_NUMBER!,
});

export const notify = createNotify({
  templates: [myTemplate],
  canals: { sms: smsCanal },
});
```

## Configuration

### `TwilioSmsAdapterOptions`

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `client` | `TwilioClient` | Yes | Twilio client (`twilio(accountSid, authToken)`) |
| `from` | `string` | Yes | Twilio phone number in E.164 format (e.g. `+15005550006`) |

## Template SMS factory

Your template's `sms` factory must return an `SmsMessage`:

```ts
const otpTemplate = createTemplate({
  id: "otp",
  sms: (args: { phone: string; code: string }) => ({
    to: args.phone,
    body: `Your verification code is: ${args.code}`,
  }),
});
```

```ts
await notify.send({
  templateId: "otp",
  canalId: "sms",
  args: { phone: "+14155552671", code: "482917" },
});
```
