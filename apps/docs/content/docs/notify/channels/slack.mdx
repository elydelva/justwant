## Overview

`slack` is not provided out-of-the-box — there is no `createSlackCanal()` helper. The `SlackMessage` type and `SlackCanal` type exist, but you must implement the canal using `createCanal()`.

## Usage

```ts title="lib/notify.ts"
import { createCanal } from "@justwant/notify";
import type { SlackCanal } from "@justwant/notify";

const slackCanal: SlackCanal = createCanal({
  kind: "slack",
  adapter: {
    async send(message) {
      await fetch(process.env.SLACK_WEBHOOK_URL!, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ text: message.text, blocks: message.blocks }),
      });
    },
  },
}) as SlackCanal;
```

## Message type

| Field | Type | Description |
|-------|------|-------------|
| `text` | `string` | Fallback text (shown in notifications and when blocks are not supported) |
| `blocks` | `unknown[]` | Optional Slack Block Kit blocks |

## Slack Incoming Webhook setup

1. Go to your Slack workspace → **Apps** → **Incoming Webhooks**.
2. Enable Incoming Webhooks and add a new webhook to a channel.
3. Copy the webhook URL and set it as `SLACK_WEBHOOK_URL` in your environment.

```bash
# .env
SLACK_WEBHOOK_URL=https://hooks.slack.com/services/T.../B.../...
```

## Registering the canal

```ts title="lib/notify.ts"
import { createNotify } from "@justwant/notify";

export const notify = createNotify({
  templates: [...],
  canals: {
    slack: slackCanal,
  },
});
```

## Example: sending a Slack notification

```ts
const deployAlert = defineTemplate({
  id: "deploy-alert",
  canal: "slack",
  render: ({ service, version }: { service: string; version: string }) => ({
    text: `Deployed ${service} v${version}`,
    blocks: [
      {
        type: "section",
        text: { type: "mrkdwn", text: `*${service}* deployed — version \`${version}\`` },
      },
    ],
  }),
});

await notify.send({
  templateId: "deploy-alert",
  canalId: "slack",
  args: { service: "api", version: "1.4.2" },
});
```
